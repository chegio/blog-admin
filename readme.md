## templates

> Common placeholder

| 占位符 | 中文 | 实例 |
| --- | --- | --- |
| <%= name %> | 模型名 | category |
| <%= pascal-name %> | 帕斯卡模型名 | Category |
| <%= const-name %> | 常量名 | CATEGORY |
| <%= plural-name %> | 复数 | categories |
| <%= pascal-plural-name %> | 帕斯卡模型复数 | Categories |
